
Servidor em java e cliente em python
Porta padrão: 4444

AVISOS IMPORTANTES:
- Primeiramente deve-se compilar o código em java para iniciar o servidor e depois
compilar o código em python (2.7) para gerar o cliente;

- Evite o uso de acentos ou caracteres especiais nas mensagens pois os mesmos
não serão reconhecidos;

- Para se enviar as mensagens através do cliente deve-se usar aspas para 
que o python possa reconhecer como tipo string;

- Para finalizar o programa, tanto pelo cliente quanto pelo servidor, só é necessário
o envio da mensagem "Tchau!".
